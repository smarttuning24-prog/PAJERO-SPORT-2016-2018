GENERAL INFORMATION


OUTLINE OF CHANGE


As the tyre pressure monitoring system (TPMS) has been employed, the service procedure has been set as follows. The other service procedures are the same as before. <Vehicles for GCC, Mexico and Eastern Europe Countries>

TYRE PRESSURE MONITORING SYSTEM (TPMS)


When the tyre pressure becomes under the specified value, the TPMS function warns the driver of that state by illuminating the TPMS warning lamp.

SYSTEM CONFIGURATION




OPERATION



  • The signals from the TPMS transmitter are received by the KOS&OSS-ECU.
  • KOS&OSS-ECU processes input signals from each TPMS transmitter as well as vehicle speed signals from the ABS-ECU or ASTC-ECU. When the road tyre pressure is low, KOS&OSS-ECU sends a warning signal causing the TPMS warning lamp to be illuminated. When the TPMS has problems, KOS&OSS-ECU sends a warning signal causing the TPMS warning lamp to be flashed*.
    note *: Change to continuous illumination after flashing for approximately 1 minute.
  • For 3 seconds after the ignition switch is turned to the "ON" position, KOS&OSS-ECU illuminates the TPMS warning lamp to check any breaks in the TPMS warning lamp circuit.
  • By connecting the M.U.T.-III to the diagnosis connector, data stored in KOS&OSS-ECU (data of tyre pressure and tyre pressure sensor ID, the alarm status and warning history, etc.) can be displayed and tyre pressure sensor ID can be registered.
    note If the TPMS transmitter is replaced, register the ID codes of all the TPMS transmitters again using the M.U.T.-III.
