Code No. C1901: Vehicle speed signal


caution If there is any problem in the CAN bus lines, an incorrect diagnosis code may be set. Prior to this diagnosis, diagnose the CAN bus lines (Refer to GROUP 54C - CAN Bus Diagnostics Table ).

DIAGNOSTIC FUNCTION


KOS&OSS-ECU receives the wheel speed information from ABS-ECU or ASTC-ECU via the CAN-bus line. Although KOS&OSS-ECU receives the information that the wheel is not currently rotated from ABS-ECU or ASTC-ECU, if the TPMS transmitter sends the information that the wheel is rotated, this code is set.

PROBABLE CAUSES


The most likely causes for this diagnosis code to be set is:

  • Damaged wiring harness and connector
  • Malfunction of TPMS transmitter
  • Malfunction of KOS&OSS-ECU
  • Malfunction of ABS-ECU or ASTC-ECU

DIAGNOSIS PROCEDURE

STEP 1. M.U.T.-III CAN bus diagnosis.
Use the M.U.T.-III to diagnose the CAN bus lines.

Q. Is the check result normal?
Go to Step 3.
Repair the CAN bus lines (Refer to GROUP 54C - CAN Bus Diagnostics Table ). On completion, go to Step 2.

STEP 2. Diagnosis code recheck after resetting CAN bus lines.
Recheck if diagnosis code No. C1901 is set.
(1)Turn the ignition switch from the "LOCK" (OFF) to "ON" position.
(2)Check if the diagnosis code is set.
(3)Turn the ignition switch to the "LOCK" (OFF) position.

Q. Is the diagnosis code set?
Go to Step 3.
The procedure is complete.

STEP 3. Using M.U.T.-III other system diagnosis code.
Check if the diagnosis code is set in the ABS-ECU or ASTC-ECU.

Q. Is the diagnosis code set?
Carry out the troubleshooting for ABS-ECU (Refer to GROUP 35B - Troubleshooting) or ASTC-ECU (Refer to GROUP 35C - Troubleshooting ). Then go to Step 5.
Go to Step 4.

STEP 4. Check the TPMS transmitter acceleration.
Check the acceleration value sent by each TPMS transmitter by performing the transmitter check using the M.U.T.-III (Refer to ) when the vehicle is stationary.

  • Item 31: Acceleration, Tyre1
  • Item 32: Acceleration, Tyre2
  • Item 33: Acceleration, Tyre3
  • Item 34: Acceleration, Tyre4
  • Item 35: Acceleration, Tyre5
OK:
    Acceleration value: 5G or less

Q. Is the check result normal?
Replace the KOS&OSS-ECU (Refer to GROUP 42B - KOS&OSS-ECU), and register ID codes (Refer to ). Then go to Step 5.
Replace the TPMS transmitter (Refer to ), and register ID codes (Refer to ). Then go to Step 5.

STEP 5. Check whether the diagnosis code is set again.
Recheck if the diagnosis code is set.
(1)Turn the ignition switch from the "LOCK" (OFF) to "ON" position.
(2)Check if the diagnosis code is set.
(3)Turn the ignition switch to the "LOCK" (OFF) position.

Q. Is diagnosis code No. C1901 set?
Start over at Step 1.
The procedure is complete.
