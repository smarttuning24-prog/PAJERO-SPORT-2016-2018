note

  • *3: This diagnosis code will remain in memory and the SRS warning lamp will be switched off when the system returns to normal condition.

  • *5: This diagnosis code will be erased automatically when the system returns to its normal condition. Then the SRS warning lamp will go out.
  • *6: Though the SRS does not have emergency call function, this diagnosis code is set.
