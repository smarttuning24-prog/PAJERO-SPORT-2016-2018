DISASSEMBLY SERVICE POINT

<<A>> BALL BEARING REMOVAL

1.Use special tool remover (MD998801), remove the ball bearing.
2.Press the rear output shaft with a press and remove the ball bearing.

REASSEMBLY SERVICE POINTS

>>A<< BALL BEARING INSTALLATION

1.Check the installation direction of the ball bearing, and put it on the rear output shaft.

2.Use the special tools to install the ball bearing.

  • Installer(100) (MD998813)
  • Installer adapter(44) (MD998821)
3.Press the rear output shaft with a press and install the ball bearing.

>>B<< SYNCH SPRING/SYNCH KEY INSTALLATION

Install two synch springs. When installing the springs, Make sure that steps of front and rear spring s are positioned on a synch key, but not on the same key.
