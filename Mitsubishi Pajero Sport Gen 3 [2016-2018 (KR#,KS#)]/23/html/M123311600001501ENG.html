DISASSEMBLY

1. BRAKE DISC SET NO. 1 (B1) REMOVAL

Remove the flange, 4 discs and 4 plates.
note Check the number of disc. (Refer to BRAKE DISC AND INTERMEDIATE SHAFT INSPECTION-INSPECT BRAKE DISK SET No. 1 (B1) )

2. BRAKE PISTON NO. 1 & BRAKE CYLINDER NO. 1 REMOVAL

1.Align the special tool Adapter (MB992320) with the centre to only push the brake piston return spring sub-assembly.

2.Using the special tools and a screwdriver, remove the hole snap ring from the transmission case.

  • Base (MB992317)
  • Cylinder (MB992318)
  • Centre bolt (MB992319)
  • Flange bolt (MB992322)

3.Remove the brake piston return spring sub-assembly and the brake piston No. 1 & brake cylinder No. 1.

3. BRAKE DISC SET NO. 2 (B2) REMOVAL

1.Set the special tool Adapter (MB992321) on the brake piston return spring sub-assembly.

2.Using the special tools and a screwdriver, remove the hole snap ring from the transmission case.

  • Base (MB992317)
  • Cylinder (MB992318)
  • Centre bolt (MB992319)
  • Flange bolt (MB992322)

3.Remove the flange.

4.Remove the 4 discs, 3 plates and flange.
note Check the number of disc. (Refer to BRAKE DISC AND INTERMEDIATE SHAFT INSPECTION-INSPECT BRAKE DISK SET No. 2 (B2) )

5.Remove the brake piston return spring sub-assembly No. 2.

4. BRAKE PISTON NO. 2 & BRAKE CYLINDER NO. 2 REMOVAL

Remove the brake piston No. 2 and brake cylinder No. 2 from the transmission case.

5. PLATE STOPPER SPRING REMOVAL

Remove the plate stopper spring from the transmission case.

6. INTERMEDIATE SHAFT REMOVAL

1.Remove the hole snap ring from the transmission case.

2.Remove the intermediate shaft with the one-way clutch inner race, one-way clutch No. 3, RR planetary ring gear flange sub-assembly & RR planetary ring gear, thrust bearing race No. 7, thrust needle roller bearing and thrust bearing race No. 8 from the transmission case.
3.

Remove the one-way clutch inner race, one-way clutch No. 3, RR planetary ring gear flange sub-assembly & RR planetary ring gear, thrust bearing race No. 7, thrust needle roller bearing and the thrust bearing race No. 8 from the intermediate shaft.

REASSEMBLY


RACE AND BEARING SPECIFICATIONS

1. INTERMEDIATE SHAFT INSTALLATION

1.Apply ATF to the thrust bearing race No. 7, thrust needle roller bearing and the thrust bearing race No. 8 and install them to the intermediate shaft.

2.Install the one-way clutch inner race, one-way clutch No. 3, RR planetary ring gear flange sub-assembly & RR planetary gear to the intermediate shaft.

3.Install the intermediate shaft, RR planetary ring gear flange sub-assembly & RR planetary ring gear, one-way clutch No. 3 and one-way clutch inner race to the transmission case.

4.
caution If the hole snap ring is not installed in the correct direction, the automatic transmission may not work properly.
Check that the snap ring is installed in the correct direction.


5.
caution Align the opening of the hole snap ring with the position shown in the illustration.
Install the hole snap ring in the groove.

2. PLATE STOPPER SPRING INSTALLATION

Install the plate stopper spring to the transmission case.

3. BRAKE PISTON & BRAKE CYLINDER (NO. 2) INSTALLATION
caution Align the hole of the brake piston No. 2 & brake cylinder No. 2 with the position shown in the illustration.

Install the brake piston No. 2 & brake cylinder No. 2 to the transmission case as shown in the illustration.

4. BRAKE DISK SET NO. 2 (B2) INSTALLATION

1.Install the brake piston return spring sub-assembly No. 2 to the transmission case.


2.
caution Check the number and order of the discs, the plates and the flange.
Install the 4 discs, 3 plates and flange to the transmission case as shown in the illustration.
note Install the discs, plates and flange following the order shown in the illustration.

3.Install the flange to the transmission case.

4.Set the special tool Adapter (MB992321) on the brake piston return spring sub-assembly.



5.
caution Align the opening of the hole snap ring with the position shown in the illustration.
Using the special tools and a screwdriver, install the hole snap ring in the groove.

  • Base (MB992317)
  • Cylinder (MB992318)
  • Centre bolt (MB992319)
  • Flange bolt (MB992322)

5. BRAKE PISTON NO. 1 & BRAKE CYLINDER NO. 1 INSTALLATION

1.
caution Align the hole of the brake piston No. 1 & brake cylinder No. 1 with the position shown in the illustration.

2.Install the brake piston return spring sub-assembly to the transmission case.

3.Align the special tool Adapter (MB992320) with the centre to only push the brake piston return spring sub-assembly.

  • Centre bolt (MB992319)
  • Flange bolt (MB992322)
