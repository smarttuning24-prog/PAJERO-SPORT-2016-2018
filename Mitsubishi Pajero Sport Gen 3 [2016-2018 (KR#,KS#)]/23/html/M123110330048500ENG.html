DIAGNOSIS FUNCTION


WARNING INDICATOR

If a malfunction relating to the A/T system occurs, the symbol (A) will be displayed on the information screen in the multi-information display.
Check if the diagnosis code is set when the symbol (A) continues being displayed in the information screen in the multi information display.
note Only when the inhibitor switch is defective, symbol "A" will flash in the selector lever position indicator.
Symbol (B) (on the multi-information display) depends on the AT fluid temperature, which A/T fluid temperature sensor informs of. When the symbol (B) is displayed in the information screen in the multi information display, the A/T fluid temperature is high. (When the A/T fluid temperature sensor 1 detects the fluid temperature of approx. 136°C or higher, symbol (B) is turned on. When it drops below approximately 131°C, symbol (B) is turned off automatically. Meanwhile, when the A/T fluid temperature sensor 2 detects the fluid temperature of approx. 138°C or higher, symbol (B) is turned on. When it drops below approximately 128°C, symbol (B) is turned off automatically.)
note When A/T fluid reaches a high temperature, the shift program for the D range will be modified to lower the fluid temperature. It is control to lower temperature of A/T fluid, and this phenomenon is not trouble. If temperature of A/T fluid falls, A/T returns to normal movement.

METHOD OF READING THE DIAGNOSIS CODE
Use the M.U.T.-III to read the diagnosis code. (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points .)

METHOD OF ERASING THE DIAGNOSIS CODE
Use the M.U.T.-III to erase the diagnosis code. (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points .)
