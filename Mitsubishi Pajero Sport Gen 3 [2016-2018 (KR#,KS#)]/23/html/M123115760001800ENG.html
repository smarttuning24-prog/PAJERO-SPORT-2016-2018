Code No.P0722: Output Shaft Speed Sensor System (No Signal), P077C: Output Shaft Speed Sensor System (Low Circuit), P077D: Output Shaft Speed Sensor System (High Circuit)


PROBABLE CAUSES


OPERATION


The output shaft speed sensor detects the speed of rear ring gear, and sends the pulse signal to A/T-ECU.

DIAGNOSIS CODE SET CONDITIONS



  • While the engine is running, if any output shaft speed sensor pulse signal is not detected even when the input shaft speed sensor pulse signal is detected, code No.P0722 is set.
  • While the engine is running, when a status continues for 1 second with which the monitor voltage from output shaft speed sensor 1 is less than standard value, code No.P077C is set.
  • While the engine is running, when a status continues for 0.5 second with which the monitor voltage from input shaft speed sensor 1 is higher than standard value, code No.P077D is set.

PROBABLE CAUSES



  • Malfunction of output shaft speed sensor
  • Malfunction of rear ring gear
  • Damaged harness wires and connectors
  • Malfunction of the A/T-ECU

DIAGNOSIS PROCEDURE

STEP 1. Measure the resistance at AT-ECU connector.
Disconnect the connector, and measure the resistance between terminal PGB+ and PGB- line at the wiring harness side.
OK: 2 Ω or less

  • Disconnect the connector, and measure the resistance between terminal PGB- and body earth at the wiring harness side.

  • Disconnect the connector, and measure the resistance between terminal PGB- and body earth at the transmission assembly side.
