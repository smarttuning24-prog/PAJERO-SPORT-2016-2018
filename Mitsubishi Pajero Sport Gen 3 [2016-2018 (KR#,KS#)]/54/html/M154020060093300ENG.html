DIAGNOSIS CODE CHART


caution
  • During diagnosis, a diagnosis code associated with other system may be set when the ignition switch is turned on with connector(s) disconnected. On completion, confirm all systems for diagnosis code(s). If diagnosis code(s) are set, erase them all.
  • If the combination meter needs to be replaced as a result of troubleshooting, the current driving distance and elapsed days must be entered into the meter after the replacement in order to be used for service reminder function. Therefore, read "Integrated mileage for reminder," "Integrated days for reminder," "Mileage until Extra reminder," "Months until Extra reminder," and "Current schedule" from the meter before the replacement using the special function of M.U.T.-III, and note them. For the operation method of M.U.T.-III, refer to . If "Integrated mileage for reminder" or "Integrated days for reminder" cannot be read from the meter using M.U.T.-III, use the following method.

    1. As for the driving distance for check warning, use the driving distance displayed on the multi information display.
    2. As for the elapsed days for check warning, calculate the number of elapsed days from the delivery date to the customer (service reminder function start date) and current date.
