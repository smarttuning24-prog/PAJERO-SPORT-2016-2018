Reversing sensor system


Operation


The reversing sensor system measures the distance to the obstacle around your vehicle. Depending on the distance, the system warns the driver visually <vehicles with blind spot warning> and aurally by using the display in the combination meter and the buzzer.


"Corner sensor" information on the combination meter (multi-information display) <Vehicles with blind spot warning>




When there is an obstacle at a corner of the vehicle, the direction to the obstacle is displayed by the corner sensor indicator on the multi-information display of the combination meter, and then the buzzer integrated in the clearance warning control unit sounds.

Clearance warning control unit


The clearance warning control unit calculates the distance to an obstacle by using ultrasonic sensors when the vehicle is driven rearwards (reversed). Then the control unit activates the corner sensor indication on the multi-information display and the buzzer (incorporated in the clearance warning control unit) as applicable.

Fail-safe and diagnosis functions


The clearance warning control unit always monitors input and output signals. When the system detects a system fault, it will flash the indicator on the sonar switch and disable the reversing sensor system.
In addition, the clearance warning control unit has the following functions to facilitate system checks:

  • Setting a diagnosis code
  • Displaying service data
You can read the all items above by using the M.U.T.-III.
note On the M.U.T.-III, the reversing sensor system is referred to as "CORNER SENSOR/SONAR".
