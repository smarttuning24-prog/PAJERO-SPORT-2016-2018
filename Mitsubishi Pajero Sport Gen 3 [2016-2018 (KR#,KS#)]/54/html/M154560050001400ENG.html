DIAGNOSIS FUNCTION


HOW TO READ DIAGNOSIS CODE
Use the M.U.T.-III to read diagnosis code (Refer to GROUP 00 - Diagnosis Function).

HOW TO ERASING DIAGNOSIS CODE
Use the M.U.T.-III to erasing diagnosis code (Refer to GROUP 00 - Diagnosis Function).

CHECK OF FREEZE FRAME DATA
To detect fault and set the diagnosis code, the e-Call control unit obtains data before the confirmation of the diagnosis code and data when the diagnosis code is confirmed, and then stores the current condition of the e-Call control unit. By analysing each data from M.U.T.-III, the troubleshooting can be performed more efficiently. Display items of the freeze frame data are as follows.
