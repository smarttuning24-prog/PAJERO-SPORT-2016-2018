HEADLAMP AUTOMATIC LEVELLING MOTOR DRIVE TEST


With this inspection, the headlamp automatic levelling motor incorporated in the headlamp assembly is forcibly activated and checking the motor for normal operation is possible.

DRIVE TEST PROCEDURE (WHEN THE M.U.T.-III IS USED)


1.Connect the M.U.T.-III to the diagnosis connector (For how to connect the M.U.T.-III, refer to GROUP 00 - Diagnosis function ).
2.Turn the ignition switch to "ON" position. (Do not start up the engine.)
3.Turn ON the headlamp switch.
4.Select the M.U.T.-III screen menu as follows.
(1) Select "AFS/ACL/Leveling" displayed on the "System selection" screen.
(2) Select "Actuator test".
(3) Select "Actuator".
(4) The headlamp beam moves downward if "20%" is set for the "Value select" and upward if "80%" is set for the "Value select".
5.The drive test is completed.

DRIVE TEST PROCEDURE (WHEN THE M.U.T.-III IS NOT USED)


1.Turn the hazard warning lamp switch to the ON position.
2.Within 10 seconds after turning the ignition switch from OFF to the ON position (do not start up the engine), turn the headlamp switch from OFF to ON 5 times repeatedly. When the operation is successfully completed, the headlamp automatic levelling warning is displayed on the multi information display in the combination meter. If not displayed, turn the ignition switch OFF and perform the operation again.
3.Within 30 seconds after the headlamp automatic levelling warning is displayed, turn the hazard warning lamp switch OFF.
4.The headlamp beam moves downward for approximately 3 seconds and upward for 3 seconds. Then the beam returns to the position applicable to the present vehicle posture.
5.The drive test is completed.
