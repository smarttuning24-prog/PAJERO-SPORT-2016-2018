Code No.B1702: Transponder read error


caution Whenever the ECU is replaced, ensure that the input and output signal circuits are normal.

OPERATION
The key reminder switch (ignition key ring antenna) supplies the ignition key with power, and the ignition key sends the key ID to the ETACS-ECU. ETACS-ECU receives the key ID with the key ring antenna, and judges if the ignition key is the registered one.

TROUBLE JUDGEMENT

  • Diagnosis code No.B1702 is set when there are other ignition keys or anything that interferes with the communication (things that generate radio waves such as magnets and an air-cleaning device that has a power plug) near the ignition key that is going to start the engine.
  • If the key ID is not sent to the ETACS-ECU even when the ignition switch is turned to the ON position, diagnosis code No.B1702 is set.
note Diagnosis code No.B1702 is always set along with MPI system diagnosis code No.P0513.

PROBABLE CAUSES

  • Interference of the key ID
  • Malfunction of the key ring antenna (antenna wire)*
    note *: Code No. B1722 will be set if a trouble occurs in the key ring antenna.
  • Malfunction of ignition key
  • Malfunction of ETACS-ECU

DIAGNOSIS PROCEDURE

STEP 1. Check whether diagnosis code No.B1722 is set.

Q. Is diagnosis code No.B1722 set?
Refer to .
Go to Step 2.

STEP 2. Check the ignition key inserted in the key cylinder for interference.

Q. Are there other ignition keys or anything that interferes with the communication (things that generate radio waves such as magnets and an air-cleaning device that has a power plug) near the ignition key inserted in the key cylinder?
Move away or remove other ignition keys or things that interfere with the communication (things that generate radio waves such as magnets and a air-cleaning device that has a power plug) near the ignition key. Then, go to Step 4.
Go to Step 3.

STEP 3. Check with the spare key that has already been registered.
Check that the engine starts using the spare key that has already been registered.

Q. Is diagnosis code No.B1702 set?
Replace the ETACS-ECU (Refer to ). After registering the ID codes, go to Step 4.
Replace the ignition key for which diagnosis code No.B1702 is set, and register the ID codes again (Refer to ). After registering the ID codes, go to Step 4.

STEP 4. Check whether the diagnosis code is set again.
(1)Turn the ignition switch to the "LOCK" (OFF) position once, and then turn it to the ON position again.
(2)On completion, check that diagnosis code No.B1702 is not set again.

Q. Is diagnosis code No.B1702 set?
Return to Step 1.
The diagnosis is complete. If another diagnosis code is set, refer to Diagnosis Code Chart .
