HOW TO INITIALISE LEARNT VALUE FOR THE ELECTRIC PARKING BRAKE ACTUATOR


caution If you fail to initialise learnt value, the tension sensor initialised tension value, which the electric parking brake ECU stores, deviates from the actual value. This may cause diagnosis code C2250 or C2251 to be set although the system functions normally. Just in case, it may lead to an excessive cable tension, thus damage the actuator.
If you do any of the following operations, you have to initialise learnt value for the electric parking brake actuator (tension sensor) prior to driving:

  • Replacement of electric parking brake actuator
  • The electric parking brake is release manually.
1.Park the vehicle on a level surface.
2.Place the selector lever to the "P" position, release the parking brake, and then turn the ignition switch OFF.
3.Connect the M.U.T.-III and turn ON the ignition switch (Refer to GROUP 00 - Precautions Before Service, M.U.T.-III ).
4.Select "EPB", "Special Function" and "Reset" in that order on the System Selection Screen.
5.
caution Do not operate the electric parking brake switch while the initialisation is in progress.
Execute "Tension sensor 0 point reset".
6.Initialise learnt value for the electric parking brake actuator under the following steps:
(1) Pull up the electric parking brake switch to apply the parking brake.
Check that the brake warning lamp in the combination meter comes on.
(2) Push down the electric parking brake switch to release the parking brake.
Check that the brake warning lamp in the combination meter goes out.
7.Select "EPB" -> "Data List" on the System Selection Screen. Then check that the following items satisfy the standard values (Refer to ).

  • Item No. 6: Tension 1 input value
  • Item No. 7: Tension 2 input value
  • Item No. 44: Tension
note If not, repeat the operations from step 2.
8.Turn OFF the ignition switch, and then disconnect the M.U.T.-III.
