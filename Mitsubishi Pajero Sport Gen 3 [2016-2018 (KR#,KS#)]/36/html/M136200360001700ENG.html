Code No. C2250: Actuator malfunction


caution
  • If there is any problem in the CAN bus lines, an incorrect diagnosis code may be set. Prior to this diagnosis, diagnose the CAN bus lines (Refer to GROUP 54C - CAN Bus Diagnostics Table ).
  • Whenever ECU is replaced, ensure that the CAN bus lines are normal.
  • When you replace the electric parking brake actuator, use the M.U.T.-III to initialise learnt value for the actuator (Refer to ).
  • If you fail to initialise learnt value, the tension sensor initialised tension value, which the electric parking brake ECU stores, deviates from the actual value. This may cause diagnosis code C2250 to be set although the system functions normally.
  • When the electric parking brake is released manually, diagnosis code No. C2250 may be set.

OPERATION


The electric parking brake ECU regulates the parking brake by activating the electric parking brake actuator (electric motor).

DIAGNOSIS CODE SET CONDITIONS


The diagnosis code will be set under any of the following conditions:

  • Actuator malfunction A: The system detects that the electric motor fails or the reduction gear is locked.
  • Actuator malfunction B: The system detects that the electric motor or the reduction gear rotates freely.
  • Actuator malfunction C: The electric parking brake is not applied or released completely.
  • Actuator malfunction D: The parking brake cable(s) seized or broken
  • Actuator malfunction E: The ECU detects that the clutch mechanism in the electric parking brake actuator fails.

PROBABLE CAUSES



  • Parking brake cable seized or broken
  • Parking brake lining improperly adjusted (excessive clearance) or worn
  • The electric parking brake actuator fails

    • The electric motor fails
    • Reduction gear locked or rotates freely
    • Clutch mechanism fails
  • Malfunction of the electric parking brake ECU

DIAGNOSIS PROCEDURE

STEP 1. Parking brake cable check
Check the parking brake rear cables for proper installation, seizure or breakage.

Q. Is the check result normal?
Go to Step 2.
Repair or replace the parking brake rear cable.

STEP 2. Parking brake lining check
Check the shoe and lining assembly (LH, RH) for proper installation and thickness, and adjust the clearance if necessary.

Q. Is the check result normal?
Go to Step 3.
Adjust or replace the shoe and lining assembly.

STEP 3. M.U.T.-III data list
(1)Check the following data list (Refer to ).

  • Item No.35: Actuator fail A
  • Item No.36: Actuator fail B
  • Item No.37: Actuator fail C
  • Item No.38: Actuator fail D
  • Item No.39: Actuator fail E
(2)Check the data list, and take a note which item is marked as "Fail".
(3)Erase the diagnosis code.
(4)Turn the ignition switch to the LOCK (OFF) position.
(5)Turn the ignition switch to the ON position.
(6)Place the selector lever to the "P" position, depress the brake pedal and then operate the electric parking brake switch.
(7)Check the data list again as described in step (1).
OK: All items are marked as "Normal"

Q. Is the check result normal?
Go to Step 4.
Replace the electric parking brake actuator (Refer to ).

STEP 4. Check whether the diagnosis code is set again.
Check again if the diagnosis code is set.
(1)Erase the diagnosis code.
(2)Turn the ignition switch to the LOCK (OFF) position.
(3)Turn the ignition switch to the ON position.
(4)Check if the diagnosis code is set.

Q. Is diagnosis code No. C2250 set?
Replace the electric parking brake ECU (Refer to ).
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
