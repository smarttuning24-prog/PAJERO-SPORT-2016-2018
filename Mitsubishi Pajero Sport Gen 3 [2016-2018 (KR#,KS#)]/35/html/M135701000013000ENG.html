Code No. C1200: FR wheel speed sensor circuit
Code No. C1205: FL wheel speed sensor circuit
Code No. C1210: RR wheel speed sensor circuit
Code No. C1215: RL wheel speed sensor circuit


caution
  • If there is any problem in the CAN bus lines, an incorrect diagnosis code may be set. Prior to this diagnosis, diagnose the CAN bus lines (Refer to GROUP 54C - Troubleshooting ).
  • Whenever ECU is replaced, ensure that the CAN bus lines are normal.
  • When the hydraulic unit assembly (integrated with ASTC-ECU) is replaced, always carry out the calibration of the steering wheel sensor and the G sensor (Refer to ).

OPERATION



  • A toothed rotor (for wheel speed detection) generates a voltage pulse as it moves across the pickup field of each wheel speed sensor.
  • The wheel speed sensor transmit the frequency of the pulses to the ASTC-ECU.

DIAGNOSIS CODE SET CONDITIONS


The ASTC-ECU monitors the voltage fluctuation in each wheel speed sensor circuit. If ASTC-ECU detects the open or short circuit in the circuit, it will set a diagnosis code.

PROBABLE CAUSES


Current trouble
  • Damaged harness wires and connectors
  • Malfunction of the wheel speed sensor
  • Malfunction of the hydraulic unit assembly (integrated with ASTC-ECU)
  • Noise interference
Past trouble
  • Carry out diagnosis with particular emphasis on wiring harness and connector failures between ASTC-ECU and the wheel speed sensor. For diagnosis procedures, refer to How to treat past trouble (GROUP 00 - How to Use Troubleshooting/Inspection Service Points ).

DIAGNOSIS PROCEDURE

STEP 1. M.U.T.-III CAN bus diagnosis
Use the M.U.T.-III to diagnose the CAN bus lines.

Q. Is the check result normal?
Go to Step 3.
Repair the CAN bus line (Refer to GROUP 54C - Troubleshooting ). On completion, go to Step 2.

STEP 2. Diagnosis code recheck after resetting CAN bus lines

Q. Is diagnosis code No. C1200, C1205, C1210 or C1215 set?
Go to Step 3.
This diagnosis is complete.

STEP 3. M.U.T.-III data list
Check the following service data (Refer to ).

  • (When code No. C1200 is set) Item No. 01: FR wheel speed sensor
  • (When code No. C1205 is set) Item No. 02: FL wheel speed sensor
  • (When code No. C1210 is set) Item No. 03: RR wheel speed sensor
  • (When code No. C1215 is set) Item No. 04: RL wheel speed sensor

Q. Is the check result normal?
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).
Go to Step 4.

STEP 4. Measure the voltage at the ASTC-ECU connector.
(1)Connector check: ASTC-ECU connector and wheel speed sensor connectors
(2)Disconnect the connector and measure at the wiring harness side.
(3)Ignition switch: "ON"
(4)Measure the voltage between the power supply terminal as well as signal terminal of each wheel speed sensor and body earth.
