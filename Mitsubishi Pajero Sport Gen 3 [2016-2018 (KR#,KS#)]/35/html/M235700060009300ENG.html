FORWARD COLLISION MITIGATION SYSTEM (FCM) AND ULTRASONIC MISACCELERATION MITIGATION SYSTEM ON/OFF SWITCH

The forward collision mitigation system (FCM) and Ultrasonic misacceleration Mitigation System ON/OFF switch is installed at the instrument panel left lower side. You can switch the FCM (on and off) and the warning timing of the forward collision warning ("FAR", "MIDDLE" and "NEAR") by operating this switch.

How to switch on and off the FCM



The FCM is toggled between on and off when the forward collision mitigation system (FCM) and Ultrasonic misacceleration Mitigation System ON/OFF switch is pressed for 3.0 seconds or more. When the FCM is turned off, the FCM OFF indicator lamp (on multi-information display) will illuminate.
When any of the following conditions is met, the FCM cannot be turned on (a visual warning will appear on the multi-information display). Note that the message does not appear when the ASC is off.

  • Transfer position: 4LLc
  • Rear differential lock: ON
  • Hill descent control: ON
  • ASC: OFF
note The ultrasonic misacceleration Mitigation System (UMS) will also be switched on and off together with the FCM.

How to switch the warning timing of the forward collision warning

The warning timing of the forward collision warning is toggled when the forward collision mitigation system (FCM) and Ultrasonic misacceleration Mitigation System ON/OFF switch is pressed for less than 3.0 seconds with the FCM activated. When the FCM is toggled, the system will inform the driver of it by the combination meter LCD display.

Fail-safe and self-diagnosis functions


The FCM-ECU constantly monitors the input and output signals. If the system detects a malfunction, it will send a failure signal to the combination meter and prohibit the FCM control under circumstances. The combination meter will provide a visual and audible warning to inform the driver of it. The FCM-ECU has the following functions for easier system checks.

  • Diagnosis code set
  • Service data output
  • Freeze frame data output
All the above items can be diagnosed using M.U.T.-III.

Radar adjustment


If the FCM-ECU is replaced or removed and installed, adjust the radar by using the M.U.T.-III and special tool.
note For how to adjust the radar, refer to the WORKSHOP MANUAL.
