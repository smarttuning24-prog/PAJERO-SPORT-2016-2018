Code No. C1014: FL wheel speed sensor performance
Code No. C101F: FR wheel speed sensor performance
Code No. C102A: RL wheel speed sensor performance
Code No. C1035: RR wheel speed sensor performance


caution
  • If there is any problem in the CAN bus lines, an incorrect diagnosis code may be set. Prior to this diagnosis, diagnose the CAN bus lines (Refer to GROUP 54C - Troubleshooting ).
  • Whenever ECU is replaced, ensure that the CAN bus lines are normal.
  • When the hydraulic unit assembly (integrated with ASTC-ECU) is replaced, always carry out the calibration of the steering wheel sensor and the G sensor (Refer to ).

OPERATION



  • A toothed rotor (for wheel speed sensor) generates a voltage pulse as it moves across the pickup field of each wheel speed sensor.
  • The wheel speed sensor transmit the frequency of the pulses to the ASTC-ECU.

DIAGNOSIS CODE SET CONDITIONS


ASTC-ECU monitors the signals from each wheel speed sensor while the vehicle is being driven. If any fault below is found in these sensor signals, ASTC-ECU will set the relevant diagnosis code.

  • Irregular change in the wheel speed sensor signal

PROBABLE CAUSES


Current trouble
  • Malfunction of the wheel speed sensor
  • Excessive gap between the wheel speed sensor and the rotor (for wheel speed detection)
  • Malfunction of the wheel speed sensor installation
  • Malfunction of the wheel bearing
  • Malfunction of the front axle hub assembly
  • Malfunction of the Axle shaft bearing
  • Deformation of the rotor (for wheel speed detection)
  • Damaged wiring harness and connectors
  • Malfunction of the hydraulic unit assembly (integrated with ASTC-ECU)
Past trouble
  • Carry out diagnosis with particular emphasis on wiring harness and connector failures between ASTC-ECU and the wheel speed sensor. For diagnosis procedures, refer to How to treat past trouble (GROUP 00 - How to Cope with Intermittent Malfunctions ).

DIAGNOSIS PROCEDURE

STEP 1. M.U.T.-III CAN bus diagnosis
Use M.U.T.-III to diagnose the CAN bus lines.

Q. Is the check result normal?
Go to Step 3.
Repair the CAN bus line (Refer to GROUP 54C - Troubleshooting ). On completion, go to Step 2.

STEP 2. Diagnosis code recheck after resetting CAN bus lines

Q. Is diagnosis code No. C101F, C1014, C1035 or C102A set?
Go to Step 3.
The procedure is complete.

STEP 3. M.U.T.-III data list
Check the following data list (Refer to ).

  • (When code No. C1014 is set) Item No. 01: FL wheel speed sensor
  • (When code No. C101F is set) Item No. 02: FR wheel speed sensor
  • (When code No. C102A is set) Item No. 03: RL wheel speed sensor
  • (When code No. C1035 is set) Item No. 04: RR wheel speed sensor

Q. Is the check result normal?
Go to Step 9.
Go to Step 4.

STEP 4. Harness and connector check
(1)Check the wheel speed sensor (FL, FR, RL or RR).
(2)Check the ASTC-ECU connector.
(3)Check the wiring harness and connector between the ASTC-ECU connector and the wheel speed sensor (FL, FR, RL or RR).

Q. Is the check result normal?
Go to Step 5.
Repair the defective connector or the wiring harness. Then go to Step 10.

STEP 5. Check for wheel speed sensor installation
Refer to .

Q. Is the check result normal?
Go to Step 6.
Repair the wheel speed sensor. Then go to Step 10.

STEP 6. Check for wheel speed sensor as a single unit
Refer to .

Q. Is the check result normal?
Go to Step 7.
Replace the wheel speed sensor (Refer to ). Then go to Step 10.

STEP 7. Check for rotor (for wheel speed detection) installation
<Front>
  • Refer to GROUP 26 - Front Axle Hub Assembly Removal and Installation <2WD>.
  • Refer to GROUP 26 - Driveshaft Assembly Removal and Installation <4WD>.
<Rear>
  • Refer to GROUP 27 - Axle Shaft Assembly Disassembly and Reassembly .

Q. Is the check result normal?
Go to Step 8.
Replace the rotor (for wheel speed detection). Then go to Step 10.

STEP 8. Front axle hub assembly and wheel bearing or axle shaft bearing check

  • Check the front axle hub assembly (Refer to GROUP 26, Front Axle Hub Assembly Removal and Installation ).
  • Check the wheel bearing (Refer to GROUP 26, On-vehicle Service ).
  • Check the axle shaft bearing (Refer to GROUP 27, On-vehicle Service ).

Q. Is the check result normal?
Go to Step 9.
Replace the front axle hub assembly or wheel bearing or axle shaft bearing. Then go to Step 10.

STEP 9. Check whether the diagnosis code is set again.
(1)Turn the ignition switch to the "ON" position.
(2)Turn the ignition switch to the "LOCK" (OFF) position.
(3)Drive the vehicle at 15 km/h or more.
note The ABS warning lamp does not turn OFF in some cases unless the vehicle runs at 15 km/h or higher.

Q. Is diagnosis code No. C101F, C1014, C1035 or C102A set?
Replace the hydraulic unit assembly (integrated with ASTC-ECU) (Refer to ).
The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to Cope with Intermittent Malfunction ).

STEP 10. Check whether the diagnosis code is set again.
(1)Turn the ignition switch to the "ON" position.
(2)Turn the ignition switch to the "LOCK" (OFF) position.
(3)Drive the vehicle at 15 km/h or more.
note The ABS warning lamp does not turn OFF in some cases unless the vehicle runs at 15 km/h or higher.

Q. Is diagnosis code No. C101F, C1014, C1035 or C102A set?
Replace the hydraulic unit assembly (integrated with ASTC-ECU) (Refer to ).
This diagnosis is complete.
