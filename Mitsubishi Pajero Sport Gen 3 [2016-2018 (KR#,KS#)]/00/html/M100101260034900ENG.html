INJECTOR IDENTIFICATION CODE REGISTRATION PROCEDURE <4D5, 4N1>


caution
  1. If the injector identification code is not registered, the engine warning lamp goes on and diagnosis code No. P1626 is logged.
  2. Failure to register the injector identification code correctly will cause rough idling, abnormal noise, and emission deterioration.

PURPOSE

Because individual injectors have different injection characteristics, the engine-ECU corrects injection time for each cylinder to improve injection accuracy.
For this reason, when the injector or engine-ECU is replaced, injector correction data must be registered afterwards in the engine-ECU using the Multi Use Tester III (M.U.T.-III).
Correction data is converted into an identification code consisting of 30 alphanumeric characters and printed on the injector connector.

REGISTRATION PROCEDURE (DURING ENGINE ECU REPLACEMENT)
caution To prevent damage to M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position <except vehicles with KOS> (or put the power supply mode is OFF <vehicles with KOS>) before connecting or disconnecting M.U.T.-III.


1.After the ignition switch is at the "LOCK" (OFF) position <except vehicles with KOS> (or put the power supply mode is OFF <vehicles with KOS>), connect the M.U.T.-III to the diagnosis connector.
2.Turn the ignition switch to the "ON" position <except vehicles with KOS> (or put the power supply mode is ON <vehicles with KOS>).
3.Select "MPI/GDI/DIESEL" from System select Screen of the M.U.T.-III.
4.Select "Special Function" from MPI/GDI/DIESEL Screen.
5.Select "Inj.ID Writing (Exchanging ECU)" from Special Function Screen.
6.Select "Injector ID Read&Save" from Inj.ID Writing (Exchanging ECU) Screen to read data from the current engine-ECU and save the data.
note If the injector identification code cannot be read and cannot be stored from the engine ECU that is not replaced yet, register the identification code in the same procedures as the injector replacement.
7.Turn the ignition switch to the "LOCK" (OFF) position <except vehicles with KOS> (or put the power supply mode is OFF <vehicles with KOS>).
8.Replace the engine-ECU.
9.Turn the ignition switch to the "LOCK" (OFF) position <except vehicles with KOS> (or put the power supply mode is OFF <vehicles with KOS>).
10.Select "MPI/GDI/DIESEL" from System select Screen of the M.U.T.-III.
11.Select "Special Function" from MPI/GDI/DIESEL Screen.
12.Select "Inj.ID Writing (Exchanging ECU)" from Special Function Screen.
13.Select "Saved Injector ID Writing " from Inj.ID Writing (Exchanging ECU) Screen to write the data, which was saved previously, to the engine-ECU.
note When the registration is completed, the lit engine warning lamp begins to blink.
14.Execute small injection quantity learning (Refer to ).
15.Confirm that the engine warning lamp is off. Confirm also that the diagnosis code is not set.

REGISTRATION PROCEDURE (DURING INJECTOR REPLACEMENT)
note If the injector identification code cannot be read when the engine-ECU is replaced, input the identification code in the following procedures:
caution To prevent damage to M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position <except vehicles with KOS> (or put the power supply mode is OFF <vehicles with KOS>) before connecting or disconnecting M.U.T.-III.


1.After the ignition switch is at the "LOCK" (OFF) position <except vehicles with KOS> (or put the power supply mode is OFF <vehicles with KOS>), connect the M.U.T.-III to the diagnosis connector.
2.Turn the ignition switch to the "ON" position <except vehicles with KOS> (or put the power supply mode is ON <vehicles with KOS>).
3.Select "MPI/GDI/DIESEL" from System select Screen of the M.U.T.-III.
4.Select "Special Function" from MPI/GDI/DIESEL Screen.
5.Select "Inj.ID Writing (Exchanging INJ.)" from Special Function Screen.
6.From the Inj.ID Writing (Exchanging INJ.) Screen, select the injector to which the identification code is registered.
Content
 M.U.T.-III registered items
No. 1 injector only registered
 No. 1 Injector ID Writing
No. 2 injector only registered
 No. 2 Injector ID Writing
No. 3 injector only registered
 No. 3 Injector ID Writing
No. 4 injector only registered
 No. 4 Injector ID Writing
All injector registered
 All Injectors ID Writing
caution Even if the number of the cylinder to be registered does not match the actual injector mounting location, registration ends normally. Specify the cylinder number correctly.

7.Register the injector identification code. Input the identification code printed on the upper plane of the injector.
note The identification code is divided into eight parts. Input the identification code into the specified parts.
8.Push "OK" to register it.
note When the registration is completed, the lit engine warning lamp should begin to blink.
9.Execute small injection quantity learning (Refer to ).
note When the injector is replaced, executing the write operation clears the values of small injection quantity learning.
10.Confirm that the engine warning lamp is off. Confirm also that the diagnosis code is not set.
