REMOVAL SERVICE POINTS

<<A>> ROCKER ARMS AND SHAFT ASSEMBLY REMOVAL
1.When the rocker arm is removed from the rocker shaft, tag them to show the installation location for the reinstallation of the rocker arm, the piston arm assembly and so on.

2.<Exhaust side only>
Set special tool Lash adjuster holder (MD998443) to prevent the lash adjuster coming from free and falling to the floor.
caution If the lash adjuster is re-used, clean the lash adjuster. (Refer to lash adjuster inspection .)

<<B>> CAM POSITION SENSING CYLINDER REMOVAL

Install the camshaft sprocket and the camshaft sprocket bolt to the camshaft. Using the special tools End yoke holder (MB990767) and Pins (MD998719), prevent the cam position sensing cylinder from rotating and remove it.

INSTALLATION SERVICE POINTS

>>A<< CAMSHAFT, RIGHT / CAMSHAFT, LEFT INSTALLATION

caution Use care to prevent confusion of the right and left bank camshafts.
1.Apply engine oil to the camshaft journals and cams and then install the camshafts.
note The left bank camshaft is identified by a slit 4 mm wide at the rear end of the camshaft.
2.Install the cylinder head, placing the dowel pin of the camshaft upward.

>>B<< CAM POSITION SENSING CYLINDER INSTALLATION
1.Mate the groove on the camshaft rear end face properly with the protrusion of the cam position sensing cylinder so that the cam position sensing cylinder can be installed.

2.Install the camshaft sprocket and the camshaft sprocket bolt to the camshaft. Using the special tools End yoke holder (MB990767) and Pins (MD998719), prevent the cam position sensing cylinder from rotating and tighten it to the specified torque.

>>C<< LASH ADJUSTER INSTALLATION
caution If the lash adjuster is re-used, clean the lash adjuster. (Refer to lash adjuster inspection .)

Fit the lash adjuster onto the rocker arm without allowing diesel fuel to spill out. Fit special tool Lash adjuster holder (MD998443) to prevent the lash adjuster coming from free and falling to the floor.

>>D<< ADJUSTING SCREW INSTALLATION

Install provisionally the screw to rocker arm, insert it so that end of the screw is flush with the edge of the rocker arm or projects slightly 3 mm.

>>E<< ROCKER ARMS AND SHAFT / ROCKER SHAFT CAP INSTALLATION

1.Place each rocker arm shaft with the side having the oil holes of the illustrated shape facing down (toward the cylinder head).
2.Check that the oil holes on each rocker arm shaft are in the positions shown in the illustration.
note The inlet rocker arm shafts have four bolt holes each.
note The exhaust rocker arm shafts have eight bolt holes each.
3.According the tags showing the reinstallation locations, install the rocker arms and all other parts on each rocker arm shaft.



4.When installing the rocker arm shaft, check that the identification marks are at the positions shown in the illustration.
5.Tighten the inlet rocker arm shaft bolts so the specified torque of 31 ± 3 N·m.
6.Tighten the exhaust rocker arm shaft bolts so the specified torque of 13 ± 1 N·m
note Pay attention to the installation direction of the rocker shaft cap at the exhaust side.
7.Remove the special tool lash adjuster holder (MD998443).

>>F<< CAMSHAFT OIL SEAL INSTALLATION


1.Use special tools MD998713 and MD998777 to install the camshaft oil seal.
2.Screw the camshaft sprocket bolt until the special tool touches the end face of the cylinder head.

3.Oil seal shall be pressed fit to position.

>>G<< WATER OUTLET FITTING GASKET INSTALLATION



Install the water outlet fitting gasket so that the tab is positioned as shown in the illustration.

>>H<< ROCKER COVER INSTALLATION

Tighten the rocker cover to the specified torque in the sequence shown in the illustration.
