REMOVAL SERVICE POINTS

<<A>> ROCKER ARMS AND ROCKER ARM SHAFT REMOVAL

Attach the special tool Lash adjuster holder (MD998443) to retain the lash adjusters during removal operation.

INSTALLATION SERVICE POINTS

>>A<< LASH ADJUSTER INSTALLATION

Install the lash adjuster into the rocker arm, being careful not to spill the diesel fuel it contains. Retain the installed lash adjuster with the special tool Lash adjuster holder (MD998443).

>>B<< ROCKER ARMS AND ROCKER ARM SHAFT/ROCKER SHAFT SPRING INSTALLATION

1.Install the rocker arm shaft by loosely fastening the bolts such that any of the inlet rocker arms does not push the valve.
2.Fit each rocker shaft spring from above and make it at right angles to the plug guide.
note Install the rocker shaft springs before installing the exhaust valve rocker arms and rocker arm shaft.

3.Remove the special tool Lash adjuster holder (MD998443) retaining each lash adjuster.
4.Make sure that the flat portion on the rocker arm shaft faces in the direction shown in the drawing.

>>C<< CAMSHAFT OIL SEAL INSTALLATION

Use the special tool camshaft oil seal installer (MD998713) to drive each oil seal into position in the cylinder head.
