INSTALLATION SERVICE POINTS

>>A<< IDLER GEAR/BALANCER SHAFT DRIVE HOUSING AND GEAR ASSEMBLY/CRANKSHAFT GEAR INSTALLATION

1.After wiping off the dirt on the crankshaft gear as shown in the figure using a rag, degrease the areas.
note Degrease them to prevent drop in the friction coefficient of the pressed area, which is caused by oil adhesion.
2.

Check that the timing marks on the gears are aligned. Then install the components.

>>B<< THRUST PLATE INSTALLATION

1.Apply the appropriate and minimum amount of engine oil to the thrust plate, as shown in the illustration.
2.Install all of these so that the thrust plate face where the engine oil is applied can face toward the balancer shaft drive housing and gear assembly.

>>C<< DOWEL PIN INSTALLATION
The dowel pin should protrude 5.75 ± 0.15 mm from the end of the balancer unit.

>>D<< BALANCER UNIT/BALANCER UNIT BOLT INSTALLATION

1.Install the washer to the balancer unit bolt so that the washer chamfered side faces as shown.
2.Apply the minimum amount of engine oil to the balancer unit bolt threads and the seating surface.
3.
caution The crankshaft should always be turned in a clockwise direction.
Set the No. 1 cylinder at top dead centre.

4.Install the lower crankcase so that the timing mark on the balancer unit should face as shown.
5.Tighten the balancer unit bolt to the specified torque in the order shown in the figure.
Tightening torque: 20 ± 1 N·m

6.
caution
  • When the tightening angle is smaller than the specified tightening angle, the appropriate tightening capacity cannot be secured.
  • When the tightening angle is larger than the specified tightening angle, remove the bolt to start from the beginning again according to the procedure.
Use special tool angle gauge (MB991614) to tighten them an additional 60°.
7.Loosen the balancer unit bolt fully.
8.Tighten the balancer unit bolt to the specified torque again in the order shown in the figure.
Tightening torque: 20 ± 1 N·m
9.Use special tool angle gauge (MB991614) to tighten them an additional 60°.
