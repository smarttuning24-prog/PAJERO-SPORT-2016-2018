DISASSEMBLY SERVICE POINT

<<A>> CYLINDRICAL ROLLER BEARING REMOVAL

Using special tool Bearing remover (MD998801), support the cylindrical roller bearing and remove it.

REASSEMBLY SERVICE POINTS

>>A<< CYLINDRICAL ROLLER BEARING INSTALLATION

1.Check the installation direction of the cylindrical roller bearing, and put it on the counter shaft.

2.
caution Never push the sealing area and outer race at the press fit.
Using special tools to install the cylindrical roller bearing.

  • Installer cap (MD998812)
  • Installer-100 (MD998813)
  • Installer adapter (34) (MD998817)

>>B<< SHAFT SNAP RING INSTALLATION

1.Select a snap ring to secure clearance of standard value between the cylinder roller bearing and counter shaft.
Standard value: 0 - 0.1 mm
2.Install the selected snap ring.
