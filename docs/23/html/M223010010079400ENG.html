GENERAL INFORMATION


Automatic transmission R8AWG is mounted for 2WD, and V8AWG is mounted for 4WD, respectively. These automatic transmissions have been improved as below, compared with the previous models.

  • Super select 4WD-II is mounted for 4WD.
  • The new automatic transmission fluid (MITSUBISHI MOTORS GENUINE ATF-MA1) is adopted. This ATF lowers viscosity in the actual operating temperature range and reduces drag loss, improving the fuel economy. With the reduced drag loss, the low temperature starting performance of the engine is improved.
  • The A/T fluid filler pipe and level gauge is abolished.
  • A selector lever with a sport mode has been adopted.
  • A paddle shift mechanism has been adopted as optional equipment.
  • Electric shift lock mechanism has been adopted.
