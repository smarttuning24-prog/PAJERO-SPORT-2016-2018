DISASSEMBLY

1. SECOND BRAKE CYLINDER REMOVAL

Remove the second brake cylinder from the one-way clutch No. 1.

2. ONE-WAY CLUTCH NO. 1 REMOVAL

Remove the one-way clutch No. 1 and the planetary carrier thrust washer No. 1 from the one-way clutch inner race sub-assembly.

3. BRAKE PISTON RETURN SPRING SUB-ASSEMBLY NO. 3 REMOVAL


1.Set the special tools Spring compressor (MB992315), as show in the illustration.
2.
caution
  • When compressing the brake piston return spring sub-assembly No. 3, be careful not to brake it.
  • Be careful not to expand the shaft snap ring too much.
Using the special tool and a screwdriver, remove the hole snap ring.

3.Remove the brake piston return spring sub-assembly No. 3 from the second brake piston.

4. SECOND BRAKE PISTON REMOVAL

While pushing the second brake piston by hand, apply compressed air 392kPa into the oil passage as shown in the illustration and remove the second brake piston from the second brake cylinder.

5. O-RING REMOVAL

Remove the 2 O-rings from the second brake piston.

1.Coat 2 new O-rings with ATF.
2.Install the O-rings to the brake piston No. 1.

2. BRAKE PISTON NO. 1 INSTALLATION

1.Coat the brake cylinder No. 1 and the brake piston No. 1 with ATF.
2.
caution Be careful not to damage the O-rings.
Install the brake piston No. 1 to the brake cylinder No. 1.

3. O-RING INSTALLATION

4. SECOND BRAKE PISTON INSTALLATION

1.Coat the second brake cylinder and the second brake piston with ATF.
2.
caution Be careful not to damage the O-rings.
Install the second brake piston to the second brake cylinder.

5. BRAKE PISTON RETURN SPRING SUB-ASSEMBLY NO. 3 INSTALLATION

Install the brake piston return spring sub-assembly No. 3 to the second brake piston.

6. HOLE SNAP RING INSTALLATION

1.Set the special tools Spring compressor (MB992315), as show in the illustration.

2.
caution
  • When compressing the brake piston return spring sub-assembly No. 3, be careful not to brake it.
  • Be careful not to expand the shaft snap ring too much.
Using the special tool and a screwdriver, Install the hole snap ring.

7. ONE-WAY CLUTCH NO. 1 INSTALLATION

1.Coat the planetary carrier thrust washer No. 1 with ATF.
2.Install the planetary carrier thrust washer No. 1 to the one-way clutch inner race sub-assembly.
3.Install the one-way clutch No. 1 to the one-way clutch inner race sub-assembly.

8. SECOND BRAKE CYLINDER INSTALLATION

Install the second brake cylinder to the one-way clutch No. 1.
