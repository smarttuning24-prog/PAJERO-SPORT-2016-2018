DISASSEMBLY

1. SHAFT SNAP RING REMOVAL <2WD>
Remove the shaft snap ring and then remove the thrust bearing race and the thrust needle roller.

2. BRAKE DISC SET NO. 4 (B4) REMOVAL

Remove the 2 flanges, 8 discs and 7 plates.
note Check the number of disc. (Refer to RR PLANETARY GEAR ASSEMBLY INSPECTION-INSPECT BRAKE DISK SET No. 4 (B4) )

3. RR PLANETARY GEAR ASSEMBLY REMOVAL

Remove the RR planetary gear assembly with the thrust bearing race No. 9, thrust needle roller with race bearing and thrust needle roller bearing.

4. LOW/REVERSE BRAKE RETURN SPRING SUB-ASSEMBLY REMOVAL

1.Set the special tools as shown in the illustration, and compress the spring.

  • Spring compressor body (MD998903)
  • Retainer (MD998924)
2.
caution Be careful not to expand the shaft snap ring too much.
Using special tools, remove the shaft snap ring.

3.Remove the low/reverse brake return spring sub-assembly.

5. INNER BRAKE PISTON (NO. 4) REMOVAL

1.While pushing the inner brake piston No. 4 by hand, apply compressed air (392 kPa) into the oil passage as shown in the illustration and remove the inner brake piston No. 4 from the transmission case.
note When applying compressed air, shut a oil passage of the transmission case as shown in the illustration.

2.Remove the inner brake piston No. 4, brake reaction sleeve and reverse brake piston.

3.Remove the 2 O-rings from the inner brake piston No. 4.

4.Coat the sliding surface of the transmission case with ATF.

5.
caution Be careful not to damage the O-rings.
Install the inner brake piston No. 4, brake reaction sleeve and reverse brake piston to the transmission case.

2. LOW/REVERSE BRAKE RETURN SPRING SUB-ASSEMBLY INSTALLATION

1.Install the low/reverse brake return spring sub-assembly.

2.Set the special tools as shown in the illustration, and compress the spring.

  • Spring compressor body (MD998903)
  • Retainer (MD998924)
3.
caution Be careful not to expand the shaft snap ring too much.
Using a screwdriver, install the shaft snap ring in the groove.

3. SELECTION OF FLANGE
Adjustment of brake pack clearance and select the brake flange No. 4. (Refer to OIL PUMP AND CENTRE PLANETARY GEAR ASSEMBLY INSPECTION-INSPECT PACK CLEARANCE OF BRAKE (B4) )

4. RR PLANETARY GEAR ASSEMBLY INSTALLATION

1.Apply ATF to the thrust bearing race No. 9, thrust needle roller with race bearing and thrust needle roller bearing and install them to the RR planetary gear assembly.

2.Install the RR planetary gear assembly to the transmission case.

5. BRAKE DISC SET NO. 4 (B4) INSTALLATION
caution Check the number and order of the flanges, the discs and the plates.


Install the 2 flanges, 8 discs and 7 plates to the transmission case as shown in the illustration.
note Install the flanges, the discs and the plates following the order shown in the illustration.

6. THRUST BEARING RACE INSTALLATION <2WD>

1.Select a thrust bearing race which will make the gear spline piece axial play equal the standard value.
Standard value: 0.02 - 0.12 mm

2.Install a thrust bearing race of the same size as having been used.

7. SHAFT SNAP RING INSTALLATION <2WD>
Install the thrust bearing race and the thrust needle roller. Then install the shaft snap ring.
