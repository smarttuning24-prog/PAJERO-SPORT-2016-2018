DIAGNOSIS FUNCTION


WARNING INDICATOR

SELECTOR LEVER POSITION INDICATOR

If a malfunction occurs in the A/T system, the selector lever position indicator in the multi-information display will flash.
note
  • Only when the inhibitor switch is defective, symbol "A" will flash in the selector lever position indicator.
  • In the manual mode, the current driving range indicator will flash.

A/T FLUID TEMPERATURE WARNING LAMP
The illumination/turning OFF of A/T fluid temperature indicator lamp is judged by the fluid temperature from A/T fluid temperature sensor 2. When the A/T fluid temperature indicator lamp is illuminated, it indicates that the A/T fluid temperature is high. (The indicator lamp illuminates when the fluid temperature is at approximately 140°C or higher, and goes out at approximately 130°C or lower.)

METHOD OF READING THE DIAGNOSIS CODE
Use the M.U.T.-III to read the diagnosis code (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points).

METHOD OF ERASING THE DIAGNOSIS CODE
Use the M.U.T.-III to erase the diagnosis code (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points).
