Code No. C1403: Mode switch failure


OPERATION



  • When the transfer-ECU measures 0 V at the drive mode-selector position which the driver selects, it will determine that the appropriate position is turned on. When the ECU measures system voltage at the other selector positions, it will determine that the other positions are turned off.
    System voltage is applied to the drive mode-selector when the ignition switch is turned ON.
  • System voltage is applied to the transfer-ECU when the drive mode-selector is in the "2H," "4H," "4HLc" or "4LLc." The transfer-ECU judges that the drive mode-selector is in the "2H," "4H," "4HLc" or "4LLc" when the system voltage is applied.

DIAGNOSIS CODE SET CONDITIONS


In the case that the input signal from the drive mode-selector is open or shorted, code No.C1403 is set as the open circuit or the short circuit of the drive mode-selector system.

PROBABLE CAUSES



  • Malfunction of the drive mode-selector
  • Damaged harness wires and connectors
  • Malfunction of the transfer-ECU
