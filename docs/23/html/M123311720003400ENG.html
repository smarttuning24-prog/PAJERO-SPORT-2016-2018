DISASSEMBLY

1. BREATHER TUBE REMOVAL

1.Remove the bolt in order to remove the breather tube and breather hose from the transmission case.
SIZE: Bolt
M6 x 1.0 x 10 mm
2.Remove the O-ring from the breather tube.

2. SPEED SENSOR (INPUT) REMOVAL
caution Be careful not to damage the speed sensor.

Remove the bolt to remove the speed sensor from the transmission case.
SIZE: Bolt
M6 x 1.0 x 14 mm

3. SPEED SENSOR (OUTPUT) REMOVAL
caution Be careful not to damage the speed sensor.

Remove the bolt to remove the speed sensor from the transmission case.
SIZE: Bolt
M6 x 1.0 x 14 mm

4. TRANSMISSION CONTROL SHAFT LEVER LH REMOVAL

Remove the nut and spring washer in order to remove the transmission control shaft lever LH from the manual valve lever shaft.

5. UNION REMOVAL

1.Remove the 2 unions.
2.Remove the 2 O-rings from the unions.

6. INHIBITOR SWITCH REMOVAL

1.Using a screwdriver, pry off the washer.
2.Remove the nut to remove the washer from the inhibitor switch.

3.
caution Be careful not to damage the inhibitor switch.
Remove the bolt (seal bolt) to remove the inhibitor switch from the transmission case.
SIZE: Bolt
M8 x 1.25 x 30 mm (Seal bolt)

REASSEMBLY

1. INHIBITOR SWITCH INSTALLATION

1.Temporarily install the inhibitor switch with the new bolt (seal bolt).
SIZE: Bolt
M8 x 1.25 x 30 mm (Seal bolt)
note Tighten the bolt after adjusting the inhibitor switch to the "N" position.

2.Install the new washer with the nut.
3.Tighten the nut to the specified torque 6.9 ± 0.9 N·m.
4.Using a screwdriver, stake the washer.

5.6.As shown in the illustration, align the reference line of the inhibitor switch with the manual valve lever shaft surface.
7.Tighten the inhibitor switch mounting bolt to the specified torque 13 ± 2 N·m.
8.Adjust the manual valve lever shaft to the "P" position.

2. UNION INSTALLATION

1.Coat the new 2 O-rings with ATF, and then install it onto the unions.
2.Install the 2 unions on the transmission case to the specified torque of 30 ± 5 N·m.

3. TRANSMISSION CONTROL SHAFT LEVER LH INSTALLATION

1.Install the transmission control shaft lever LH using the washer and the nut.
2.Tighten the nut to the specified torque 16 ± 1 N·m.

4. SPEED SENSOR (OUTPUT) INSTALLATION

1.
caution Be careful not to damage the speed sensor.
Install the speed sensor to the transmission case with the bolt.
2.Tighten the speed sensor mounting bolt to the specified torque 5.4 ± 1.5 N·m.
SIZE: Bolt
M6 x 1.0 x 14 mm

5. SPEED SENSOR (INPUT) INSTALLATION

1.
caution Be careful not to damage the speed sensor.
Install the speed sensor to the transmission case with the bolt.
2.Tighten the speed sensor mounting bolt to the specified torque 5.4 ± 1.5 N·m.
SIZE: Bolt
M6 x 1.0 x 14 mm

6. BREATHER TUBE INSTALLATION

1.Install the bolt in order to install the breather tube and breather hose from the transmission case.
2.Tighten the breather tube mounting bolt to the specified torque 5.4 ± 1.5 N·m.
SIZE: Bolt
M6 x 1.0 x 10 mm
