Code No. C1430: Transfer position SW


OPERATION


System voltage is applied to the each transfer actuator from the transfer-ECU.

DIAGNOSIS CODE SET CONDITIONS



  • When the transfer changeover is not completed while driving, code No. C1430 is set for abnormality of each transfer actuator.
  • When the transfer changeover is not completed while the vehicle is stopped, code No. C1430 is set for abnormality of each transfer actuator.
  • When the abnormality of each transfer actuator is detected all the time, code No. C1430 is set.

PROBABLE CAUSES



  • Damaged harness wires and connectors
  • Malfunction of the transfer actuator
  • Malfunction of the transfer-ECU
