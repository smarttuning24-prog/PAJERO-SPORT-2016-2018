Inspection Procedure 1: The security alarm is not armed (the security indicator does not illuminate).


caution
  • Before replacing the ECU, ensure that the power supply circuit, the earth circuit and the communication circuit are normal.
  • When the ETACS-ECU of vehicles without KOS is replaced, the encrypted code of the ignition key needs to be registered to the ETACS-ECU. (If the encrypted code is not registered, the engine cannot be started. Register the encrypted code as described in Immobilizer System - How to Register Key ID .)

COMMENTS ON TROUBLE SYMPTOM


If this function does not work normally, the input signal circuits to the components below, the security indicator, the ETACS-ECU or the CAN bus line may have a problem.

  • Keyless entry transmitter
  • Key reminder switch
  • Ignition switch (ACC)
  • Door switches
  • Tailgate latch

PROBABLE CAUSES



  • Malfunction of CAN bus line
  • Malfunction of security indicator
  • Malfunction of keyless entry transmitter
  • Malfunction of the key reminder switch
  • Malfunction of door switch
  • Malfunction of tailgate latch
  • Malfunction of the KOS-ECU
  • Malfunction of the ETACS-ECU
  • Damaged harness wires and connectors

DIAGNOSIS PROCEDURE


STEP 1. M.U.T.-III CAN bus diagnostics


Use the M.U.T.-III to diagnose the CAN bus lines.

Q. Is the check result normal?

Go to Step 2.

Repair the CAN bus line. (Refer to GROUP 54C - Troubleshooting ).

STEP 2. M.U.T.-III other system diagnosis code


Check if diagnosis code is set in the ETACS-ECU, KOS-ECU <vehicles with KOS> and combination meter.

Q. Is the diagnosis code set?

YES <when the ETACS-ECU> : Troubleshoot the ETACS-ECU (Refer to ).

YES <when the KOS-ECU> : Troubleshoot the KOS (Refer to GROUP 42B - KOS ).

YES <when the combination meter> : Troubleshoot the combination meter (Refer to ).

NO : Go to Step 3.

STEP 3. Check the keyless entry transmitter.


Check that the keyless entry system works normally.

Q. Is the check result normal?

Go to Step 4.

NO <vehicles with KOS> : Troubleshoot the KOS (Refer to GROUP 42B - KOS ).

NO <vehicles without KOS> : Troubleshoot the keyless entry (Refer to GROUP 42A - keyless entry ).

Q. Is the check result normal?

YES <Normal conditions are displayed for all the items.> : Go to Step 5.

NO <Normal condition is not displayed for item No. 29.> : Troubleshoot the ETACS-ECU. Refer to Inspection Procedure 1 "The ignition switch (ACC) signal is not received" .

NO <Normal condition is not displayed for item No. 28> : Refer to Inspection Procedure 3 "The key reminder switch signal is not received" .

NO <Normal condition is not displayed for item No. 19.> : Troubleshoot the ETACS-ECU. Refer to Inspection Procedure 5 "The front door switch (driver’s side) signal is not received" .

NO <Normal condition is not displayed for item No. 48.> : Troubleshoot the ETACS-ECU. Refer to Inspection Procedure 6 "The front door switch (passenger’s side) signal is not received" .

NO <Normal condition is not displayed for item No. 49.> : Troubleshoot the ETACS-ECU. Refer to Inspection Procedure 7 "One of the door switch signals is not received" .

STEP 5. Security indicator check.


(1)Disconnect the ETACS-ECU connector, and test at the wiring harness side.
(2)Use a jumper wire to short the ETACS-ECU connector (DTHE) terminal with body earth.
(3)Check the security indicator of the combination meter illuminates.

Q. Is the check result normal?

Go to Step 7.

Go to Step 6.

STEP 6. Check of open circuit in (DTHE) line between ETACS-ECU connector and combination meter connector.


Q. Is the check result normal?

Replace the combination meter (Refer to ).

Repair the connector(s) or wiring harness.

STEP 7. Retest the system


Q. Does the security alarm work normally?

The trouble can be an intermittent malfunction (Refer to GROUP 00 - How to use Troubleshooting/inspection Service Points - How to Cope with Intermittent Malfunction ).

Replace the ETACS-ECU (Refer to ).
