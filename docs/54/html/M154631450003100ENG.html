DEALER MODE


HOW TO INITIATE THE DEALER MODE

1.Press the "A" section on the menu screen for at least five seconds.

2.The dealer mode will be initiated. Then "Dealer" screen will be displayed.

HOW TO TERMINATE THE DEALER MODE
If the operations below are done, the dealer mode will be terminated.

  • If "Back" button is selected on "Dealer" screen, the dealer mode will terminate and then return to the previous screen.

DEALER MODE ITEM
The following items can be checked or set in the dealer mode.

  • Select Model
  • Check Connectivity
  • Guideline Settings
  • Import Vehicle Type Data
  • Speaker Check
  • Version Information

Select Model

1.Select "Select Model" on the dealer mode screen.

2.The vehicle name is displayed. Select "KS0W/KR0W PAJERO SPORTS" button.

Check Connectivity

1.Select "Check Connectivity" on the dealer mode screen.


2."Check connectivity" screen will be displayed.
GPS
  • Check whether the GPS antenna is connected.
    If a GSP antenna connection is recognised, "Connection present" will be displayed. If it cannot be recognised, "No connection" will be displayed.
  • Press "GPS" button to display how the GPS signal are received on the GPS position screen.
Speed Sensor
  • Displays whether the vehicle speed signal is received.
    If the vehicle speed signal is received from the combination meter during driving, "OK" will be displayed. If it cannot be received, "---" will be displayed.
Back Sensor
  • Check whether the inhibitor switch is connected.
    If the R position signal is received when the selector lever is moved to the "R" position, "OK" will be displayed. If it cannot be received, "---" will be displayed.
Gyro Sensor
  • Displays how the gyro sensor incorporated in the multivision display is.
    If the gyro sensor is normal, "OK" will be displayed. If the gyro sensor is not abnormal, "---" will be displayed.
Illumination
  • Displays whether the illumination signal is received.
    "OK" will be displayed when the rain light sensor determines that it is night time. If it cannot be received, "---" will be displayed.
Rear Camera
  • Displays whether the rear view camera is connected.
    If the rear view camera connection is recognised, "OK" will be displayed. If it cannot be recognised, "---" will be displayed.
Steering RC(MODE)
  • Displays how "MODE" switch on the steering wheel audio remote control switch is operated.
    "OK" will be displayed when "MODE" switch on the steering wheel audio remote control switch is pressed. If it cannot be received, "---" will be displayed.
RSES
  • Displays whether the rear display is connected.
    If the rear display connection is recognised, "OK" will be displayed. If it cannot be recognised, "---" will be displayed.
Premium Amp
  • Displays whether the audio amplifier is connected.
    If the audio amplifier connection is recognised, "OK" will be displayed. If it cannot be recognised, "---" will be displayed.
Multi Camera
  • Displays whether the C-ON and CMSW line is connected between the MMES and the camera-ECU.

    1. Turn the ignition switch to the "ON" position.
    2. Operate the multi around monitor operation switch.
    3. If the system recognises that the MMES is connected with the camera-ECU on the C-ON and CMSW line, "OK" will be displayed.

Guideline Settings

1.Select "Guideline Settings" button on the dealer mode screen.


2.Draw base lines 0.5 m behind the vehicle and width +0.4 m (+0.2 m each for right and left) and align the guide line over the base lines.

Import Vehicle Type Data
1.Adds vehicle information stored in a SD card.

Speaker Check

1.Select "Speaker Check" button on the dealer mode screen.

2."Speaker Check" screen will be displayed. Press "TEST TONE" button to display, the speakers outputs from each speaker automatically.

Version Information

1.Select "Version Information" button on the dealer mode screen.

2."Version Information" screen will be displayed.
