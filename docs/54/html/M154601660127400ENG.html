SERVICE MODE


HOW TO INITIATE THE SERVICE MODE

1.Press "INFO/SET" button and "SEEK TRACK >>l" button at the same time for at least 3.5 seconds.


2.The service mode will be initiated. Then "Service" screen will be displayed.

HOW TO TERMINATE THE SERVICE MODE
If the operations below are done, the service mode will be terminated.

  • If "Back" button is selected on "Service" screen, the service mode will terminate and then return to the previous screen.

SERVICE MODE ITEM
The following items can be checked or set in the service mode.

  • Present Position Revision
  • Unit Diagnosis
  • Monitor Adjustment
  • Camera Setting
  • Versions Log Information
  • Versions Indication
  • GPS Time Setting
  • Hands Free Module
  • Radio Information
  • Car Type Setting
  • Handle Setting
  • Memory Initialization
  • Serial Number Information
  • AUX Button Constant Display
  • Update vehicle information

Present Position Revision

1.Select "Present Position Revision" button on "Service" screen.



2.Adjust the position, and then press "Set" button. Then adjust the direction, and then press "Set" button.
note If a road type must be selected, select "Highway" or "Regular Road".

3.A current location will be displayed on "Current location revised." screen.

Unit Diagnosis

Select "Unit Diagnosis" button on "Service" screen.

Communications & Connection Check

1.Select "Communications & Connection Check" button on "Unit Diagnosis" screen.

2.A network and connect line check will be initiated. The "Communications & Connection Check" screen will display how the check is in progress.


3.When the network and connect line check is finished, the screen will change to "Network/Connect Line Result" to show the check results.
note If there is "NG" or "N/A" as the check results, select "NG" on the "Network/Connect Line Result" screen. Then "NG code Indication" screen will show the NG code.
4.If "Back" is selected on "Network/Connect Line Result", the screen will return to "Unit Diagnosis" screen.

Key Check

1.Select "Key Check" button on "Unit Diagnosis" screen.

2.When a hard key on the multivision display or a steering wheel remote control switch is pressed on "Key Check" screen, its name will be displayed.
note The switch name will be displayed while the switch is being pressed.

Vehicle Signal Check

1.Select "Vehicle Signal Check" on "Unit Diagnosis" screen.

2.The current status of vehicle signal will be displayed.

  • "Speed": "ON" when the vehicle speed is 5 km/h or more, and "OFF" when the vehicle speed is 3 km/h or less.
  • "ILL": "ON" when the lighting switch is on (the tail lamps or the headlamps are ON), and "OFF" when the tail lamps or the headlamps are OFF.
  • "Shift Position R": "ON" when the shift lever <M/T> or selector lever <A/T> is at R position, and "OFF" when it is at the other position.

SD Card Check

1.Select "SD Card Check" on "Unit Diagnosis" screen.

2.When "Start" button of "Map Slot Check Results" is pressed on "SD Card Check" screen, the SD card which is inserted in the map slot will be checked.
note When "Start" button is pressed on "USER slot Check Results", the SD card which is inserted the USER slot will be checked.



3.Check result will be displayed.

  • If the check result is OK.
  • If the check result is not OK.

      • W: Writing check of a dummy file
      R: Reading check of a dummy file
      V: Reference check of a dummy file
      D: Deletion of a dummy file
  • If no SD card is detected:

Speaker Check

1.Select "Speaker Check" button on "Unit Diagnosis" screen.

2.Select a speaker to be checked, and play test tone through the speaker.
note
  • By operating the volume key, the volume can be changed.
  • During the test, only the selected speaker sounds. If "Back" is selected during the test, the test tone will disappear.

Sensor Check

1.Select "Sensor Check" button on "Unit Diagnosis" screen.



2.Press "Start" button on "Sensor Check" screen, and then follow instructions on the screen to operate the vehicle.

3.Check result will be displayed. If any of the check results is not OK, a NG code will be displayed.

Monitor Adjustment

Select "Monitor Adjustment" button on "Service" screen.

Monitor Check

1.Select "Monitor Check" button on "Monitor Adjustment" screen.

2.16-gradient greyscale pattern are displayed.

3.By touching the greyscale screen, it will be switched to a full-scale white screen.

4.By touching the full-scale white screen, it will be switched to a eight colour bars.

5.By touching the eight colour bars, it will be switched to a cross-hatch pattern. (Check whether the full-scale red frame is not misaligned)

6.By touching the cross-hatch screen, it will be switched to a monocolour vertical stripe screen.

7.By touching the monocolour vertical stripe screen, it will be switched to a full-scale black screen.

8.By touching the full-scale black screen, it will be switched to a full-scale red screen.

9.By touching the full-scale red screen, it will be switched to a full-scale green screen.

10.By touching the full-scale green screen, it will be switched to a full-scale blue screen.
11.By touching the full-scale blue screen, it will be switched back to "Monitor Adjustment" screen.

Touch Switch Confirmation

1.Select "Touch Switch Confirmation" button on "Monitor Adjustment" screen.


2.If you touch the screen, the colour of the dotted coordinate at the touched area will be changed.

Touch Switch Revision

1.Select "Touch Switch Revision" button on "Monitor Adjustment" screen.

2.Select "Set" button according to the instructions on the screen.
3.When the confirmation button is selected after the adjustment, it will be switched back to "Monitor Adjustment" screen.

Camera Setting

1.Select "Camera Setting" button on "Service" screen.


2.Draw base lines 0.5 m behind the vehicle and width +0.4 m (+0.2 m each for right and left) and align the guide line over the base lines.

Versions Log Information

Select "Versions Log Information" button on "Service" screen.

Output Log File (Navi)

1.Select "Output Log File (Navi)" button on "Log Function" screen.

2.Press "Output to Music SD Card" button on "System Log Output" screen in order to write a file.
note If no SD card is inserted in the USER slot, "Output to Music SD Card" button will not be active.
3.After the file writing is complete, restart the navigation system.

Telematics Log

1.Select "Telematics Log" button on "Log Function" screen.



2.Select "Display History" or "History Output"from " from the POI update items on "Telematics Log" screen.
note If no update history is stored, the icon above will not be active.
"Display History" is selected:
  • The last 30 items in the POI update history will be displayed.
"History Output" is selected:
  • Write the POI update history to the SD card which is inserted in the USER slot.

Service Data Log

1.Select "Service Data Log" button on "Log Function" screen.

2.The logs are displayed from the latest one.
note
  • Eleven error logs will be backed up in maximum.
  • All backed-up error logs will be erased by selecting "Delete" button.

Versions Indication


1.Version info. will be displayed by selecting "Versions Indication" button on "Service" screen.

GPS Time Setting

1."GPS Time" screen will be displayed by selecting "GPS Time Setting" button on "Service" screen.
2.Select "Set" button.

3.Set your local time on "GPS Time Setting" screen, and then select "Set" button.

Hands Free Module <Vehicles with Hands Free-ECU>

1.Select "Hands Free Module" on "Service" screen.
note For item "Hands Free Module", refer or Hands Free-ECU - Diagnosis Mode .)

Radio Information

Select "Radio Information" button on "Service" screen.

Radio Sensitivity

1.Select "Radio Sensitivity" button on "Radio Information" screen.

2."Radio Reception Status" screen will be displayed.

TMC Sensitivity

1.Select "TMC Sensitivity" on "Radio Information" screen.

2."TMC information" screen will be displayed.
note
  • Information on current received broadcast stations is displayed.
  • When "Start" button is selected, the current radio station will be diagnosed.

3.Select "Station List" on "TMC information" screen. Then select and preset one of the displayed radio stations.

4.Select "Manual Setting" on "TMC information" screen. Then preset a radio station by pressing "<" or ">".

Car Type Setting

1.Select "Car Type Setting" on "Service" screen.

2.The vehicle name is displayed. Select "4P00" button, and confirm it by pressing "Set" button.

Handle Setting

1.Select "Handle Setting" on "Service" screen.

2.Select the currently-set handle to display.
3.To change the handle setting, select the handle to be changed to, and then select "Set."
note Even when the handle is selected, the selected handle will not be set if "Back" is selected.

Memory Initialization
caution Executing "Memory Initialization" will erase the data stored by the customer. If you do so, it has only to be in agreement with the customer.

1.Select "Memory Initialization" on "Service" screen.




2.If you select "Start" on "Memory Initialization" screen, the settings such as registered locations and music server will be erased (initialised) from the memory.
3.After the memory initialisation is complete, the navigation system will restart automatically.

Serial Number Information

1.Select "Serial Number Information" on "Service" screen.

2."Serial Number Information" screen will be displayed.

AUX Button Constant Display

1.Select "AUX Button Constant Display" on "Service" screen.

2.Select either "Always Display" or "Only Display when Device Connected" on "AUX Button Constant Display" screen.

Update vehicle information

1.Select "Update vehicle information" on "Service" screen.

2."Update Vehicle INFO" screen will be displayed.
3.Select "Upload" button.
note If no update file is stored in the SD card, this button will not be active.


4.Select "Yes" button to upload the file.

5.After the upload is complete, restart the navigation system.
