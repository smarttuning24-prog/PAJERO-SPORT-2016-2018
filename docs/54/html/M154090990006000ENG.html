Camera calibration


Calibrate the camera-ECU according to the procedure below after the following operation is performed:

  • Replacement of the camera-ECU
  • Removal and installation/replacement of a camera
  • Any of the door or the tailgate, Outside rear-view mirror assembly (LH), (RH) is aligned.
  • The radiator grille assembly is deformed or loosened due to a collision accident.


How to check the image screen visually





1.
note To visually check one camera, two calibration markers (MB993003) are needed. So, you cannot check all the cameras simultaneously. Perform the visual inspection one after another.
Position the calibration markers (MB993003) as shown.
2.Press the multi around monitor operation switch on the steering wheel remote control switch (LH) to display the bird’s eye-view on the multivision display.




3.Check that the white lines, which are indicated by arrows, are shown as "straight line" on the bird’s eye-view image. (These lines should be also continuous in the same thickness.)

How to write learned value to the camera-ECU
When the camera is replaced, write a learned value to the new camera-ECU according to the procedure below.
1.Connect M.U.T.-III to the diagnosis connector (For how to connect M.U.T.-III, refer to the GROUP 00 - Diagnosis Function ).
2.Select "Write learned value(Changed ECU)" and then "Learned value Read&save" in the special functions for the multi-around monitor system to save the learned value, which is stored in the current camera-ECU, to the M.U.T.-III.
note If the learned value cannot be read, replace the camera-ECU and then calibrate the camera.
3.Replace the camera-ECU (Refer to ).
4.Select "Write learned value(Changed ECU)" and then "Saved learned value writing" in the special functions to write the learned value, which is stored in the M.U.T.-III, to the camera-ECU.
5.If any problems are not found on the screen, the calibration is complete. If any problem is found, calibrate the camera.

How to calibrate a camera
caution Do not interrupt a camera vision range while a camera-ECU is calibrated.
1.When the camera-ECU is replaced, first write a learned value to the camera-ECU.
2.Park the vehicle on a level surface.
note Prevent sun light rays from interfering with the camera, otherwise the monitor screen will be reflected in white. Ideally, the brightness of a road surface should be low (such as black).
3.One person should sit on the driver’s seat.
4.Connect M.U.T.-III to the diagnosis connector (For how to connect M.U.T.-III, refer to the GROUP 00 - Diagnosis Function).





5.Position the calibration markers (MB993003) as shown.

  • Position the calibration markers in respect to the relevant camera only.
note
  • The calibration markers should be located in the permissible range. The range is within ± 3 cm and 1° from the standard position.
6.Turn the ignition switch from "LOCK" (OFF) position to "ON" position.
7.Use the M.U.T.-III to calibrate the camera.
note When you execute the calibration, all calibration-related data will be erased from the ECU.
8.Select "Calibration" in the "special function" for the multi-around monitor system.
9.Select a camera to calibrate on the service data located to the left of the calibration screen, and then press "OK".
