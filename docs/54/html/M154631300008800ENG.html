ERROR DISPLAY


If the multivision display detects an error under the conditions below, the error will be displayed.

  • When the multivision display is activated
  • When the OPEN/TILT key is operated on the multivision display
  • When a CD/DVD is loaded/ejected
  • When a CD/DVD is played
  • When an audio file, which is stored in an SD memory card or an USB device, is played
  • When you use a hands-free system
