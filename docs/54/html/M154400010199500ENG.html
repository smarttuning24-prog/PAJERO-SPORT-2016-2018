GENERAL INFORMATION


A multivision display (6.95-inch wide display) has been adopted for the smartphone link display audio.
The feature is as below.

  • Connect smartphone and duplicate screen.
  • Connect smartphone with USB cable.
  • Remote operation from embedded big size display.
  • Able to provide latest service without big investment.
  • Relying on smartphone apps for features and services.
  • Able to catching up evolution of smartphone apps.
  • If compliant Android device is connected, Apps icon on multivision display will change to Android Auto icon. If compliant iPhone is connected, Apps icon on multivision display will change to CarPlay icon.

 Apple CarPlay is a smarter, safer way to use your iPhone in the car. Apple CarPlay takes the things you want to do with iPhone while driving and displays them on the screen of the product.
