MULTI INFORMATION DISPLAY <COLOUR LIQUID CRYSTAL DISPLAY TYPE>


In the multi information display, followings are displayed; "Drive mode indicator display", "Caution mark display", "FCM OFF indicator display", "Off road mode indicator display", "Cruise control symbol display", "Cruise control display", "Information screen/interrupt display screen", "Door-ajar warning display", "Selector lever position", "Engine coolant temperature display", "Fuel remaining display", "Odometer", "Outside temperature display", "Meter illumination control". Displayed contents can be switched by operating the multi information display switch or rheostat illumination switch.
note
  • *: Is not displayed during driving.

METER ILLUMINATION CONTROL


The brightness of the combination meter can be adjusted in eight stages (LEVEL 1 to LEVEL 8) by pushing the rheostat illumination switch when the meter is illuminated. Current illumination level will be displayed each time the rheostat illumination switch is pushed. The illumination level can be changed by operating the rheostat illumination switch according to specified procedure *. The brightness of each brightness level is as described in the table below.
note The brightness can be set separately for the daytime and nighttime. The configured information is stored in the RAM. In the initial setting of brightness, the daytime is set at LEVEL 8 and the nighttime is set at LEVEL 7.

SERVICE REMINDER FUNCTION


When the inspection interval (alert start point in table below) is due in accordance with the predetermined schedule (see table below), the service reminder indicator will be displayed when the ignition switch is turned to the ON position. However, even if the warning condition is met, the indicator does not display immediately. It illuminates next time when the ignition switch is turned ON. The impending inspection interval can be displayed by changing the LCD section by the multi information display switch. The service reminder indicator will go off when it is reset by operating the multi information display switch according to the specified procedure *, or the warning criteria (end point) shown in the table below is satisfied.
note *: For details about the multi information display switch special operation, refer to the Workshop Manual.
