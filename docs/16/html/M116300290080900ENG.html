DETONATION SENSOR CHECK


<4G6>
Check the detonation sensor circuit if self-diagnosis code, No. P0327 or P0328 is shown.
(Refer to GROUP 13A - Troubleshooting - Inspection Procedure for Diagnosis Code - Code No. P0327 Detonation Sensor Circuit Low Input ).
(Refer to GROUP 13A - Troubleshooting - Inspection Procedure for Diagnosis Code - Code No. P0328 Detonation Sensor Circuit High Input ).

<6B3>
Check the detonation sensor circuit if self-diagnosis code, No. P0327, P0328, P0332 or P0333 is shown.
(Refer to GROUP 13B - Troubleshooting - Inspection Procedure for Diagnosis Code - Code No. P0327 Right Bank Detonation Sensor Circuit Low Input ).
(Refer to GROUP 13B - Troubleshooting - Inspection Procedure for Diagnosis Code - Code No. P0328 Right Bank Detonation Sensor Circuit High Input ).
(Refer to GROUP 13B - Troubleshooting - Inspection Procedure for Diagnosis Code - Code No. P0332 Left Bank Detonation Sensor Circuit Low Input ).
(Refer to GROUP 13B - Troubleshooting - Inspection Procedure for Diagnosis Code - Code No. P0333 Left Bank Detonation Sensor Circuit High Input ).
