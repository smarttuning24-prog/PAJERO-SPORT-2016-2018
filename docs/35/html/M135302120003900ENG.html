SENSOR CALIBRATION


G SENSOR CALIBRATION


caution If the ASTC-ECU sets the diagnosis code other than C1392 and C1394 before calibration, follow the relevant troubleshooting procedure.
caution When the hydraulic unit assembly (integrated with ASTC-ECU) is replaced, always carry out the calibration of the steering wheel sensor and the G sensor.
caution Carry out the calibration under the following conditions.

  • The vehicle has one occupant (driver) only.
  • Turn the steering wheel to set the wheels in the straight-ahead positions.
  • The brake pedal is not depressed. (The stop lamp switch is OFF.)
1.Park the vehicle on a level surface.
2.Turn the ignition switch to the "ON" position.
3.Turn the ignition switch to the "LOCK" (OFF) position.
4.
caution Before connecting or disconnecting M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position.
Before setting M.U.T.-III, turn the ignition key to the "LOCK" (OFF) position.
5.Turn the ignition switch to the "ON" position.
6.Select "ABS/ASC/ASTC/WSS."
7.Select "Special function."
8.Select "Sensor calibration."
9.Select "G sensor calibration."
10.Before removing M.U.T.-III from the vehicle, turn the ignition key to the "LOCK" (OFF) position.

STEERING WHEEL SENSOR CALIBRATION


caution If any of the following operations has been carried out, follow the two operations below:

  • Front wheel alignment
  • Column switch assembly (Integrated with steering wheel sensor) is replaced or removed and installed
  • The ASTC-ECU is replaced

  1. Update the neutral position stored in the steering wheel sensor.
  2. Reset the correction value of the steering wheel angle, which is stored in the ASTC-ECU.
caution
  • If the ASTC-ECU sets the diagnosis code other than C1392 and C1394 before calibration, follow the relevant troubleshooting procedure.
1.Park the vehicle on a level surface.
2.Turn the ignition switch to the "ON" position.
3.Turn the ignition switch to the "LOCK" (OFF) position.
4.
caution Before connecting or disconnecting M.U.T.-III, always turn the ignition switch to the "LOCK" (OFF) position.
Before setting M.U.T.-III, turn the ignition key to the "LOCK" (OFF) position.
5.Turn the ignition switch to the "ON" position.
6.Select "Steering Angle Sensor."
7.Select "Special function."
8.Select "SAS Calibration."
9.Select and run "SAS Initialization." (Reusable parts only)
note
  • Since the neutral position stored in the steering wheel sensor is no longer available, "SAS Initialization" is displayed.
  • Since a new steering wheel sensor has no neutral position, "SAS Calibration" is displayed.
10.Select and run "SAS Calibration."
11.Exit "Steering angle sensor" menu, and then select "ABS/ASC/ASTC/WSS."
12.Select "Special function."
13.Select "Sensor calibration."
14.Select "SAS calibration."
15.Before removing M.U.T.-III from the vehicle, turn the ignition key to the "LOCK" (OFF) position.
