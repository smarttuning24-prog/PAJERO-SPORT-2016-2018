ABS warning lamp, Brake warning lamp


The ABS system informs the driver of the ABS system status by illuminating, extinguishing, or flashing the ABS warning lamp and brake warning lamp as follows.
ABS warning lamp
  • Turns ON when a system malfunction occurs.
Brake warning lamp
  • Turns ON when an EBD system malfunction occurs.
    note
    • Turns ON when the brake fluid level in the reservoir tank becomes the specified value or lower.
    • Turns ON when the parking brake lever is pulled and the brake is activated.
