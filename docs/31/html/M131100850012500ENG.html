TPMS TRANSMITTER CHECK OR REPLACE


After removing the tyre from the wheel, remove the TPMS transmitter (Refer to ). Then check it as follows and replace if necessary:
1.Wipe off the agent remained on the TPMS transmitter housing. In particular, remove it thoroughly around the pressure detection hole.
note Wait for a while to allow the agent to set, and then detach it.
2.If any of the following conditions is found, replace the TPMS transmitter and then register the tyre pressure sensor ID code: In addition, replace the valve, the valve core and the cap. Then install the tyre assembly to the wheel (Refer to ).

  • Damage (deformation or scratch, etc.) on the exterior of the housing
  • The agent cannot be removed thoroughly from inside the pressure detection hole.
3.If the conditions above are not found, the TPMS transmitter can be reused. Replace the valve, the valve core and the cap only. Then install the tyre assembly to the wheel (Refer to ).
note If the TPMS transmitter is reused, the pressure measurement function should be checked as follows.

  • Inflate the corresponding tyre to 250 kPa or more, and perform step 2 in the TPMS transmitter check (Refer to ).
  • Inflate the corresponding tyre to 150 kPa or less, and perform step 2 in the TPMS transmitter check (Refer to ).
If abnormal pressure difference (20 kPa or more) is observed in these checks, replace the TPMS transmitter and register the tyre pressure sensor ID code.
