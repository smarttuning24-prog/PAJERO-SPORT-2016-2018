INITIALISATION PROCEDURE FOR LEARNING VALUE IN DIESEL PARTICULATE FILTER (DPF) SYSTEM <4N1-EURO5>


PURPOSE
The engine-ECU stores the change of the DPF system caused by the various driving conditions as the learning value and uses it for assumption of the PM accumulated amount or for the acceptance/rejection criteria of the DPF system. For this reason, the learning value is necessary to be initialised when the parts related to the DPF system are replaced.

INITIALISATION PROCEDURE
1.After the ignition switch is in the "LOCK" (OFF) position <except vehicles with KOS> (or put the power supply mode is OFF <vehicles with KOS>), connect M.U.T.-III with the diagnosis connector.
2.Ignition switch is in the "ON" position <except vehicles with KOS> (or put the power supply mode is ON <vehicles with KOS>).
3.Select "MPI/GDI/DIESEL" from System select Screen of the M.U.T.-III.
4.Select "Special Function" from MPI/GDI/DIESEL Screen.
5.Select "Initialisation" from Special Function Screen.
6.Select the item on the screen of the initialisation for learning, and perform the initialisation.
