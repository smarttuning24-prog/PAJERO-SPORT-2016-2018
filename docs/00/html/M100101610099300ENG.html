CONFIGURATION DIAGRAMS


Connector symbol
 Name
 Reference page
 Description of change
A
 ENGINE COMPARTMENT
LHD

  • Due to the introduction of the washer fluid level warning lamp, washer fluid level switch (A-34) has been added.
  • Due to the introduction of the adaptive cruise control (ACC), ACC/FCM-ECU (A-123) has been added.
RHD

  • Due to the introduction of the battery current sensor, battery current sensor <Vehicles for Thailand (6B3)> (A-09) has been added.
  • Due to the introduction of the washer fluid level warning lamp, washer fluid level switch (A-34) has been added.
  • Due to the introduction of the adaptive cruise control (ACC), ACC/FCM-ECU (A-123) has been added.
D
 INSTRUMENT PANEL
 LHD

  • Due to the introduction of the adaptive cruise control (ACC), the name of the connector below has been changed:

    • "Instrument panel wiring harness and control wiring harness combination <4D5, 4N1>" (D-08) to "Instrument panel wiring harness and control wiring harness combination <4D5, 4N1> or no connection <6B3>"
  • Due to the introduction of the accessory socket, accessory socket (instrument centre panel) (D-258) connector has been added.
  • Due to the introduction of the gateway-ECU (for smartphone link display audio), the following connectors have been added.

    • Instrument panel wiring harness and smartphone link display audio sub wiring harness <Vehicles for Mexico> (D-259)
    • Gateway-ECU (for smartphone link display audio) <Vehicles for Mexico> (D-260)
    • Smartphone link display audio <Vehicles for Mexico> (D-261)
RHD

  • Due to the introduction of the adaptive cruise control (ACC), the name of the connector below has been changed:

    • "Instrument panel wiring harness and control wiring harness combination <4D5, 4N1>" (D-08) to "Instrument panel wiring harness and control wiring harness combination <4D5, 4N1> or no connection <6B3>"
  • Due to the introduction of the accessory socket, accessory socket (instrument centre panel) (D-258) connector has been added.
G
 DOOR
 LHD

  • Due to the change of the centre door lock, the following connectors have been added.

    • Power window main switch (G-20)
    • Front door wiring harness and power window switch sub wiring harness combination (G-21)
RHD

  • Due to the change of the centre door lock, power window main switch (G-20) has been added.
