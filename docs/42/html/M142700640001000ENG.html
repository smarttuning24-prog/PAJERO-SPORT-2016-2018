INSPECTION PROCEDURE 4: Vehicle-speed sensitive automatic door lock (with impact-sensitive unlock mechanism) does not work normally.


caution Before replacing the ECU, ensure that the input and output signal circuits are normal.

COMMENTS ON TROUBLE SYMPTOM


If all of the doors and the tailgate are not locked after the vehicle speed reaches 15 km/h or more with all of the doors and the tailgate closed, all the signal circuits for all the doors and the tailgate, the vehicle speed signal (which is received from the engine-ECU via CAN) or the ETACS-ECU may be defective. Alternatively, it is possible that "Automatic door lock (Speed)" has been set as "Disable" by means of the customisation function.

PROBABLE CAUSES



  • Malfunction of ETACS-ECU
  • Damaged wiring harness and connectors

DIAGNOSIS PROCEDURE


STEP 1. Checking central door unlocking operation


Check that the central door locking system works normally.

Q. Is the check result normal?

Go to Step 2.

Refer to Central Door Locking System Symptom Chart .

STEP 2. Check the customisation function.


Check that "Relock" is set for "Automatic door lock (Speed)" with the customisation function.

Q. Is the check result normal?

Go to Step 3.

Set "Relock" for "Automatic door lock (Speed)" with the customisation function (Refer to Customisation Function ).

STEP 3. M.U.T.-III CAN bus diagnostics


Use M.U.T.-III to diagnose the CAN bus lines.

Q. Is the CAN bus normal?

Go to Step 4.

Repair the CAN bus line (Refer to GROUP 54C - Troubleshooting ).

STEP 4. Check the M.U.T.-III Diagnosis code of other systems


Check if diagnosis code No. U1100 is set in SRS-ECU.

Q. Is the diagnosis code set?

Diagnose SRS (Refer to GROUP 52B - Troubleshooting ).

Go to Step 5.

STEP 5. Retest the system.


Are all of the doors and the tailgate locked after the vehicle speed reaches 15 km/h or more with all of the doors and the tailgate closed?

Q. Is the check result normal?

Intermittent malfunction (Refer to GROUP 00 - How to Use Troubleshooting/Inspection Service Points - How to Cope with Intermittent Malfunction ).

Replace ETACS-ECU (Refer to GROUP 54A - ETACS, ETACS-ECU ).
