DOOR INSIDE HANDLE PLAY ADJUSTMENT


DOOR INSIDE HANDLE KNOB PLAY CHECK

1.Check that the door inside handle play is within the standard value range.
Standard value (B):
Front door: 10.7- 25.6 mm (target value: 18.0 mm)
Rear door: 9.9 - 24.8 mm (target value: 17.2 mm)
2.If it is outside the standard value range, check the door inside handle and door latch assembly and replace them if necessary.

DOOR INSIDE HANDLE LOCK KNOB STROKE CHECK
1.Remove the door trim assembly. Refer to GROUP 52A, Door Trim .

2.Check that the stroke of the door inside handle lock knob is within the standard value range.
Standard value (B): 13.5 - 15.5 mm (target value: 14.5 mm)
3.Check the door inside handle lock knob and door latch assembly. Replace them if the stroke is out of the standard value.
